
# Node API skeleton

Combining the Node.js/ES6 libs du jour for the purpose of providing JSON APIs for apps.

* Babel/ES6 transpiling
* Express for middleware
* Jest for testing, Supertest for mocking express

## Install Dependencies

```
yarn install
```

## Testing

```
yarn test
```
